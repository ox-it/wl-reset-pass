Redefinir Senha: Resumo

O que faz

A ferramenta Redefinir Senha permite que você solicite uma nova senha do sistema, simplesmente digitando o seu endereço de e-mail.

Considerações

Este recurso só pode estará disponível para você se você tiver uma conta de convidado no sistema. Se você normalmente faz o login com um nome de usuário diferente do seu endereço de email, você pode não ser capaz de alterar sua senha usando esta ferramenta.
